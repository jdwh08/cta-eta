  <name>Task 1: Create WeatherDaemon with async polling loop</name>
  <files>src/cta_eta/data_collection/orchestration/weather_daemon.py</files>
  <action>
Create WeatherDaemon class inheriting from BaseDaemon with async run() method. The daemon:

1. **Initialization** (__init__):
   - Call super().__init__(config, logger)
   - Load NWS and Open-Meteo grid caches using factory functions from weather_grid_cache.py
   - Load stations cache using api_cta_stations.get_stations_cache()
   - Store weather collection interval from config (900 seconds = 15 minutes)

2. **Main loop** (async def run()):
   - Create httpx.AsyncClient with default timeouts
   - While self.running:
     - Call await self._collect_weather_cycle()
     - await asyncio.sleep(900)  # 15 minutes - NEVER use time.sleep()
   - On loop exit, close httpx client

3. **Collection cycle** (async def _collect_weather_cycle()):
   - Get all stations from stations cache
   - Deduplicate to unique grid points using both NWS and Open-Meteo caches:
     - Create set of unique (nws_grid_id, om_grid_id, lat, lon) tuples
     - Skip stations where grid cache returns None (cache miss, handle separately)
   - For each unique grid point:
     - Use asyncio.gather(return_exceptions=True) to query NWS + Open-Meteo in parallel
     - Pass exceptions through (don't fail entire batch on single source error)
   - Store results list with timestamps

4. **State persistence** (_get_state()):
   - Return dict with last_collection_timestamp, unique_grid_points_count

**What to avoid and WHY:**
- DON'T use time.sleep() → Blocks entire event loop, daemon becomes unresponsive to signals
- DON'T use asyncio.gather() without return_exceptions=True → NWS failure would stop Open-Meteo collection
- DON'T query all 300 stations → Use grid cache deduplication to get ~50 unique points
- DON'T create httpx client per request → Create once in run(), reuse for all calls in cycle

**Pattern to follow:**
```python
# From RESEARCH.md Pattern 4
async def run(self) -> None:
    async with httpx.AsyncClient() as client:
        while self.running:
            try:
                await self._collect_weather_cycle(client)
            except Exception as e:
                self.logger.error(f"Collection cycle failed: {e}")
            await asyncio.sleep(self.interval)  # NOT time.sleep()
```
