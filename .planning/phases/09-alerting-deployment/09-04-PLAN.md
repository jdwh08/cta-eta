    Create deploy/ directory with systemd unit files.

    **cta-train-daemon.service:**
    ```ini
    [Unit]
    Description=CTA Train Position Collection Daemon
    After=network-online.target
    Wants=network-online.target

    [Service]
    Type=simple
    User=cta-eta
    WorkingDirectory=/opt/cta-eta
    EnvironmentFile=/opt/cta-eta/.env
    ExecStart=/opt/cta-eta/.venv/bin/python -m cta_eta.data_collection.run_train_daemon
    Restart=on-failure
    RestartSec=30s
    StandardOutput=journal
    StandardError=journal
    SyslogIdentifier=cta-train-daemon

    # Give SIGTERM time for graceful shutdown before SIGKILL
    TimeoutStopSec=60s
    KillSignal=SIGTERM

    [Install]
    WantedBy=multi-user.target
    ```

    **cta-weather-daemon.service:** Same structure with:
    - Description: CTA Weather Data Collection Daemon
    - ExecStart: `...python -m cta_eta.data_collection.run_weather_daemon`
    - SyslogIdentifier: cta-weather-daemon

    **cta-alerts.service** (run by timer, not continuous):
    ```ini
    [Unit]
    Description=CTA ETA Alert Check
    After=network-online.target

    [Service]
    Type=oneshot
    User=cta-eta
    WorkingDirectory=/opt/cta-eta
    EnvironmentFile=/opt/cta-eta/.env
    ExecStart=/opt/cta-eta/.venv/bin/cta-alerts
    StandardOutput=journal
    StandardError=journal
    SyslogIdentifier=cta-alerts
    ```

    **cta-alerts.timer** (run alert check every 15 minutes):
    ```ini

    Description=CTA ETA Alert Check Timer
    Requires=cta-alerts.service

    OnUnitActiveSec=15min
    Unit=cta-alerts.service

    [Install]
    WantedBy=timers.target
    ```

  <name>Task 2: Create logrotate config and deployment README</name>
  <files>deploy/logrotate.conf, deploy/README.md</files>
  <action>
    **deploy/logrotate.conf** â€” rotate daemon JSONL log files:
    ```
    /opt/cta-eta/.daemon_state/*.jsonl {
        daily
        rotate 14
        compress
        delaycompress
        missingok
        notifempty
        copytruncate
        # copytruncate avoids SIGHUP to daemon (daemons open files without handles)
    }

    /opt/cta-eta/.daemon_state/*.log {
        daily
        rotate 7
        compress
        delaycompress
        missingok
        notifempty
        copytruncate
    }
    ```

    NOTE: Use `copytruncate` instead of `create` because the Python daemons open log files by path without releasing file handles. `create` would orphan the open file descriptor. This is important â€” document WHY in a comment.

    Also include rotation for systemd journal if logs go there (they do based on service files), but logrotate is not needed for journald (it manages its own rotation via `journald.conf`). Comment this in the file.

    **deploy/README.md** â€” deployment guide:
