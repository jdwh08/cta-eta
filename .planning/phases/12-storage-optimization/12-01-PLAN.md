  <behavior>
DriftResult dataclass fields:
  - drift_type: Literal["none", "additive", "breaking"]
  - breaking_fields: list[BreakingFieldChange] (name, old_type str, new_type str)
  - added_fields: list[AddedField] (name, type str)
  - removed_fields: list[str] (field names)
  - nullability_changes: list[NullabilityChange] (name, old_nullable, new_nullable)

classify_drift(registry_schema, observed_schema) rules:
  - Field in registry but not observed → removed_fields, drift_type=breaking
  - Field in both, same type, same nullable → no drift for this field
  - Field in both, type changed, widening (per WIDENING_PAIRS set) → no drift
  - Field in both, type changed, not widening → breaking_fields, drift_type=breaking
  - Field in both, same type, nullable changed → nullability_changes, drift_type=breaking
  - Field in observed but not registry → added_fields, drift_type=additive (if no breaking)
  - Field order differences → ignored
  - drift_type priority: breaking > additive > none

WIDENING_PAIRS (from RESEARCH.md):
  (pa.int8, pa.int16), (pa.int8, pa.int32), (pa.int8, pa.int64),
  (pa.int16, pa.int32), (pa.int16, pa.int64),
  (pa.int32, pa.int64),
  (pa.int32, pa.float64), (pa.int64, pa.float64),
  (pa.float32, pa.float64)

Registry JSON format (from RESEARCH.md):
  {
    "version": 1,
    "daemon": "train_positions",
    "updated": "2026-02-25",
    "fields": [{"name": "...", "type": "...", "nullable": bool}, ...],
    "schema_ipc_b64": "<base64 encoded IPC stream bytes>"
  }

schema_to_registry_dict(schema, daemon_name) → dict
  - fields list from schema (human-readable for git diff)
  - schema_ipc_b64 from IPC serialization (exact round-trip)
  - updated = today's date isoformat
  - version = 1

registry_dict_to_schema(data) → pa.Schema
  - decode schema_ipc_b64, open IPC stream, return schema_arrow
  - (ignores "fields" list — that is for human review only)

load_registry(path: Path) → pa.Schema | None
  - Returns None if path does not exist
  - Raises ValueError on corrupt/invalid JSON

save_registry(path: Path, schema: pa.Schema, daemon_name: str) → None
  - Creates parent directories
  - Writes JSON registry dict

bootstrap_registry(path: Path, schema: pa.Schema, daemon_name: str) → bool
  - If path exists: return False (no-op)
  - If path does not exist: save_registry(), return True

Test cases (RED → GREEN → REFACTOR cycle):

  test_classify_breaking_type_incompatible: int64→utf8 → drift_type="breaking", field in breaking_fields
  test_classify_widening_none: int32→int64 → drift_type="none"
  test_classify_nullability_breaking: required→nullable → drift_type="breaking", field in nullability_changes
  test_classify_additive_new_field: new field in observed → drift_type="additive", field in added_fields
  test_classify_field_order_ignored: same fields different order → drift_type="none"
  test_classify_none_identical: identical schemas → drift_type="none", all lists empty
  test_registry_round_trip_complex_types: timestamp[us, tz=UTC] + int64 + utf8 round-trips exactly
  test_bootstrap_creates_file: bootstrap on missing path creates file, returns True
  test_bootstrap_noop_if_exists: bootstrap on existing path is no-op, returns False
  test_load_returns_none_missing: load_registry on missing path returns None
