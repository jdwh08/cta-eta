    ## discover_journals

    Signature: `discover_journals(data_path: Path, dataset_name: str, target_date: date) -> list[Path]`

    Returns sorted list of all `journal_*.ipc` paths under:
      `{data_path}/{dataset_name}/year={YYYY}/month={MM:02d}/day={DD:02d}/`

    Returns empty list if the directory does not exist.

    Cases:
    - existing directory with 3 journal files → returns sorted list of 3 paths
    - non-existent directory → returns []
    - directory with no .ipc files → returns []
    - mixed files (.ipc + .txt) → returns only .ipc files matching journal_*.ipc

    ## read_ipc_with_repair

    Signature: `read_ipc_with_repair(path: Path) -> tuple[list[pa.RecordBatch], bool]`

    Returns (batches, was_clean):
    - batches: all valid RecordBatch objects read before any error
    - was_clean: True if read completed without ArrowInvalid (StopIteration = clean)

    Cases:
    - normal closed file (EOS marker present) → (all batches, True)
    - crash file (no EOS marker, missing close()) → (all batches, True) — pyarrow StopIteration
    - file with corrupt trailing bytes after valid batches → (valid batches, False)
    - file with corrupt header (schema unreadable) → ([], False)
    - empty file → ([], False)

    Note: Per RESEARCH.md verified behavior — pyarrow 22.0.0 raises StopIteration (not ArrowInvalid)
    for files missing the EOS marker (daemon crash without corruption). ArrowInvalid is only raised
    for actual byte corruption.

    ## ipc_reader.py

    Use the exact patterns from RESEARCH.md Pattern 1 (discover) and Pattern 2 (repair).

    For `read_ipc_with_repair`:
    ```python
    try:
        reader = ipc.open_stream(path)
    except pa.lib.ArrowInvalid:
        return [], False

    batches: list[pa.RecordBatch] = []
    while True:
        try:
            batch = reader.read_next_batch()
            batches.append(batch)
        except StopIteration:
            return batches, True
        except pa.lib.ArrowInvalid:
            return batches, False
    ```

    For `discover_journals`: use `pathlib.Path.glob("journal_*.ipc")` on the day directory,
    sort by name (chronological since filename is HHMMSS_usec).
