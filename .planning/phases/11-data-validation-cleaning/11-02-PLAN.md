    ## uploader.py

    Implement `upload_parquet(table: pa.Table, cloud_url: str, *, reprocess: bool = False) -> None` (row count derived from `len(table)` internally).

    Internally:
    1. Build pyarrow filesystem: `fs, path = fsspec.url_to_fs(cloud_url)` then `pa_fs = pafs.PyFileSystem(pafs.FSSpecHandler(fs))`.
    2. If `reprocess is True`: check if remote file already exists and log existing row count (use `pq.read_metadata(path, filesystem=pa_fs).num_rows` — wrap in try/except FileNotFoundError). Log warning "Overwriting existing file at {path} with {N} rows".
    3. Write: `pq.write_table(table, path, filesystem=pa_fs, compression="snappy")`.
    4. Verify: `expected_rows = len(table); meta = pq.read_metadata(path, filesystem=pa_fs); assert meta.num_rows == expected_rows`.
    5. Wrap steps 3-4 in `stamina.retry_context(on=Exception, attempts=3, wait_initial=1.0, wait_max=30.0, wait_exp_base=2, timeout=None)` per RESEARCH.md Pattern 6. Log each attempt number from `attempt.num` attribute.

    Module-level helper: `make_pyarrow_fs(cloud_url: str) -> tuple[pafs.PyFileSystem, str]` — returns (pa_filesystem, path_without_scheme). Used by both upload and row-count verify.

    Raise `RuntimeError` with descriptive message if all 3 attempts fail (stamina re-raises the last exception; let it propagate to compact.py which catches and triggers alert).

    ## archiver.py

    Implement two functions per RESEARCH.md Pattern 8:

    `archive_journals(journal_files: list[Path], archive_base: Path, target_date: date) -> None`
    - Creates `archive_base / f"date={target_date.isoformat()}"` directory.
    - Moves each file with `shutil.move(str(f), archive_dir / f.name)`.
    - Logs count of archived files at INFO level.

    `prune_archive(archive_base: Path, retention_days: int = 7) -> list[Path]`
    - Globs `archive_base / "date=*"` directories.
    - Parses date from dirname; deletes dirs older than retention_days via `shutil.rmtree`.
    - Returns list of pruned paths (for metrics logging).
    - Silently skips dirs with unparseable names (ValueError) or OSError on delete.

    Use `logging.getLogger(__name__)` for both modules. No print statements.

    ## compact.py

    Implements the CLI entry point and orchestration logic for the daily compaction job.

    ### CompactionMetrics dataclass

    ```python
    @dataclass
    class CompactionMetrics:
        date: str            # "2026-02-17"
        daemon: str          # "train_positions" or "weather"
        status: str          # "success" | "partial" | "failed"
        journals_found: int
        journals_repaired: int   # files where was_clean=False but had valid batches
        journals_skipped: int    # files yielding 0 batches (corrupt header)
        rows_written: int
        upload_bytes: int        # os.path.getsize of local Parquet before upload
        elapsed_seconds: float
        error: str | None = None
    ```

    ### _compact_one_daemon(daemon_name, target_date, config, *, reprocess: bool = False) -> CompactionMetrics

    Orchestration per daemon:
    1. `data_path = Path(config["storage"]["data_path"])` (default "data")
    2. `discover_journals(data_path, daemon_name, target_date)` — if empty, log warning and return metrics with status="partial", rows_written=0.
    3. Loop: `read_ipc_with_repair(path)` for each journal. Track `journals_found`, `journals_repaired` (was_clean=False but batches > 0), `journals_skipped` (batches == 0).
    4. Validate schema: compare each table's schema against `TRAIN_POSITION_SCHEMA` or `WEATHER_SCHEMA` (selected by dataset_name). Log and skip files with schema mismatch — do not concat incompatible schemas (per RESEARCH.md Pitfall 3).
    5. `pa.concat_tables(tables)` → merged table.
    6. Write local Parquet to `data/compaction/{daemon_name}/date={date}/data.parquet` (local staging file before cloud upload). Use `pq.write_table(merged, local_path, compression="snappy")`.
    7. `upload_parquet(merged, cloud_url, reprocess=reprocess)` where `cloud_url = f"{config['compaction']['cloud_url']}/{daemon_name}/date={date}/data.parquet"`.
    8. After upload success: `archive_journals(all_journal_files, archive_base, target_date)`.
    9. `prune_archive(archive_base, retention_days)`.
    10. Return metrics with status="success".

    If upload fails after 3 retries: return metrics with status="failed", error=str(exc). Do NOT archive journals. Trigger alert via `send_compaction_alert(metrics, config)`.

    ### _write_sidecar(metrics: CompactionMetrics, compaction_dir: Path) -> None

    Write `compaction_dir / f"compaction-{metrics.date}-{metrics.daemon}.json"` containing `asdict(metrics)`. Called in `finally` block so it always runs.

    ### send_compaction_alert(metrics: CompactionMetrics, config: dict) -> None

    Call the existing Phase 9 alerting system. Import `send_email_alert` from `cta_eta.monitoring.alerting` and `_build_email_config` from `cta_eta.monitoring.run_alerts`. Extract the alerting section with `alerting_cfg = config.get("alerting", {})`. If `alerting_cfg` is empty or not a dict, log a warning and return without raising. Otherwise call `email_config = _build_email_config(alerting_cfg)` to produce the provider-aware dict that `send_email_alert` expects (resolves Mailjet env vars etc.). Then call `send_email_alert(email_config, subject, body)` where subject is "CTA Compaction Failed: {metrics.daemon} {metrics.date}" and body summarizes metrics.error and journals_skipped count. Actual signature: `send_email_alert(email_config: dict, subject: str, body: str) -> bool`. NOTE: do NOT use `config["email"]` — the TOML section is `[alerting]`, not `[email]`.

    ### main() entry point

    ```python
    def main(argv=None):
        parser = argparse.ArgumentParser(prog="cta-compact", ...)
        parser.add_argument("--reprocess", type=date.fromisoformat, metavar="YYYY-MM-DD",
                            help="Re-compact a specific past date (default: yesterday)")
        args = parser.parse_args(argv)

        config = load_config()  # from cta_eta.data_collection.config
        reprocess_date = args.reprocess  # None unless --reprocess given
        target_date = reprocess_date or (date.today() - timedelta(days=1))
        is_reprocess = reprocess_date is not None

        datasets = ["train_positions", "weather"]
        compaction_dir = Path(config.get("compaction", {}).get("compaction_dir", "data/compaction"))
        compaction_dir.mkdir(parents=True, exist_ok=True)

        for daemon_name in datasets:
            t0 = time.monotonic()
            metrics = None
            try:
                metrics = _compact_one_daemon(daemon_name, target_date, config, reprocess=is_reprocess)
                metrics.elapsed_seconds = time.monotonic() - t0
            except Exception as exc:
                metrics = CompactionMetrics(
                    date=target_date.isoformat(), daemon=daemon_name, status="failed",
                    journals_found=0, journals_repaired=0, journals_skipped=0,
                    rows_written=0, upload_bytes=0, elapsed_seconds=time.monotonic() - t0,
                    error=str(exc))
                send_compaction_alert(metrics, config)
            finally:
                if metrics is not None:
                    _write_sidecar(metrics, compaction_dir)
    ```

    Module also has `if __name__ == "__main__": main()` for `python -m` invocation per systemd service.
